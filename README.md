# 🍽️ HungryBee

HungryBee is a full-stack **online food ordering web application** built using **Java, JSP, Servlets, JDBC, and MySQL**. It allows users to explore restaurants, browse menus, place orders, and manage their profiles through a simple and user-friendly interface.

## ✨ Features
- 🔐 User Authentication
- 🍴 Browse Restaurants & Menus
- 🛒 Add to Cart
- 📦 Place Orders
- 📜 Order History
- 👤 User Profile

## 🛠️ Tech Stack
- Frontend: HTML, CSS, JavaScript, JSP
- Backend: Java, Servlets, JDBC
- Database: MySQL
- Server: Apache Tomcat

## 🚀 Getting Started
1. Clone the repository.
2. Import into Eclipse.
3. Configure the MySQL database.
4. Run on Apache Tomcat.
